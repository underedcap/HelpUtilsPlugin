**Возможности**
/fly           — переключает полёт для себя
/fly 5         — устанавливает скорость полёта 0.5F

/fix           — чинит предмет в руке
/fixall        — чинит все предметы в инвентаре

/heal          — лечит себя(убирает дебаффы, и восстановит saturation == голод)

/ec            — инвентарь эндер сундука плеера
